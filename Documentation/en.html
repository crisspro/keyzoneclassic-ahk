KeyzoneClassic-AHK help

Script to use Keyzone Classic VSTI with screen readers.


    This script maybe be working with any DAW, was tested with:

  • Keyzone Classic v1.0.
  • Reaper 6.11.
  • Windows 10 x64.

Licence GPL-3.0

Source code and last releases:

https://github.com/crisspro/keyzoneclassic-ahk

Usage

  1. Run KeyzoneClassic-AHK
  2. Open Keyzone Classic in your DAW.
  3. Press the hotkeys of script.
  4. Can close the script from system tray

Hotkeys

Hotkey Description
M Open the instruments menu.
S Next instrument.
A Back instrument.
F1 Open help documentation.
CTRL+q close the script.
