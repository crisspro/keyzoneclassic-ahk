Ayuda de KeyZoneClassic-AHK

Script para usar el plugin VSTI KeyZone Classic con lectores de pantalla.


    Ha sido probado con los siguientes programas, aun cuando debiece funcionar con cualquier DAW:

  • KeyZone Classic v1.0.
  • Reaper 6.11.
  • Windows 10 x64.

Licencia GPL-3.0

Código fuente y últimas versiones:

https://github.com/crisspro/keyzoneclassic-ahk

Cómo usarlo

  1. Abre KeyZoneClassic-AHK
  2. Abre KeyZone Classic en tu secuenciador.
  3. Usa los atajos de teclado del script para comenzar a usarlo.

Atajos de teclado

    Atajos de teclado Descripción
    M Abre el menú de instrumentos.
    S Cambia al siguiente instrumento.
    A Cambia al instrumento anterior.
    F1 Abre la ayuda.
    CTRL+q Cierra el script.
